# 行先案内板システム

総社市デジタル推進課の行先案内板システムです。

## 🎯 機能

- ✅ 職員の行先一覧表示
- ✅ 色分け表示（在席=緑、外出=黄など）
- ✅ 自動更新（30秒ごと）
- ✅ スマホ対応
- ⏳ 行先編集機能（開発中）

## 🚀 使い方

### 閲覧方法

以下のURLにアクセスしてください：



https://CitySojaDXDiv.github.io/whereabouts/


### 行先の更新方法

**現在、編集機能は開発中です。**

暫定的に、以下の方法で更新してください：

1. Google Sheetsを直接編集
2. URLにアクセス：


https://docs.google.com/spreadsheets/d/1UaeXlrjj0o1Eja4uNc22uuqzV0SDM0hdjLcXS5k4h4E/edit

3. 自分の行を編集
4. 30秒以内に自動反映されます
